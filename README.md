# Getting Started
*******************************************************************************
## Step 1 Installation

### Установка необходимых библиотек
<pre>pip install -r requirements.txt</pre>

### Переход в корневую директорию
<pre>cd ./web</pre>

### Запуск локального сервера
<pre>python manage.py runserver</pre>
